# Text Elements
Equals in machine language to "go to":
the memory address of a + type size * offset ^CIPJQOGc

To walk that space you could do: ^69RDcCSO

This means it cannot grow because else it will overwrite
memory that shouldn't be touched. ^7OQDr0IR

a + typesize * offset ^mz6u5N7e

to get an specific index ^b0eSXAOO

This means that operations like getting an index, insertion at an index o deletion
at an index (which is basically an insertion) will have a complexity of O(1). ^vJMUN2nA

[] ^rcloTOcV

4 bytes ^jOcfZHzF

32 bits ^XwocTIsw
